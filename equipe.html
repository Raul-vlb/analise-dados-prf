Sobre o projeto

Este dashboard foi desenvolvido como trabalho acadêmico de análise e visualização de dados, utilizando o dataset público de incidentes registrados pela Polícia Rodoviária Federal (PRF) nas rodovias federais brasileiras em 2026.

A aplicação é inteiramente client-side (sem backend) e utiliza Vega-Lite, PapaParse e Bootstrap 5 para processamento e visualização dos dados diretamente no navegador.
